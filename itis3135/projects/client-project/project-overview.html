Project Overview: T&O Construction Website

Project Overview

Client and Project Category

For this project, I selected Category 02 – client outside of the class. My client is an individual, Rutilo De La Luz, who owns a small construction business named T&O in the Charlotte, NC area. This project falls under the category of “an individual that you know to promote their professional projects.” The goal of the website is to increase the visibility of T&O, improve communication with potential customers, and provide an easy way for visitors to request construction services.

Application Description and Purpose

The “T&O” web application is a multi-page promotional website for a Charlotte-based construction business owned by Rutilo De la Luz. Its purpose is to showcase completed roofing, siding, exterior remodeling, and interior remodeling projects, clearly communicate the range of services offered, and make it easy for potential clients to request quotes or consultations online. The site will highlight real project photos, describe the company’s capabilities, and provide simple, user-friendly forms to initiate contact and service requests.

Intended Users

  • Homeowners in the Charlotte area needing roofing, siding, exterior or interior remodeling work.
  • Property managers and landlords requiring ongoing maintenance or repair services.
  • Real estate investors needing reliable contractors for renovations.
  • Local residents searching online for trustworthy construction services.

Overview of Website Content

  • Overview of T&O and its construction services.
  • Detailed descriptions of roofing, siding, exterior remodeling, and interior remodeling services.
  • Project gallery with photos and details of completed work.
  • Testimonials or short quotes from satisfied customers.
  • Contact and “Request a Quote” forms.
  • Service area information focused on Charlotte, NC and surrounding regions.

Page Design

Home Page (index.html)

  • Purpose: Introduce T&O and highlight roofing, siding, exterior remodeling, and interior remodeling services for Charlotte-area visitors.
  • Audience/Users: All visitors, especially potential clients researching home services.
  • Content: Hero banner image, short company overview, featured services, call-to-action links to projects and quote request.
  • Data Entry: No data entry on this page.
  • Validations: Not applicable.
  • Buttons/Hyperlinks/Dropdowns: “View Services” button, “Request a Quote” button, and navigation links.
  • Actions: Buttons and links navigate to Services, Projects, and Contact pages; optional JavaScript-based rotating hero images.
  • Special Notes: Includes shared header and footer across all pages with HTML and CSS validation links in the footer.

Services Page (services.html)

  • Purpose: Present the full list of construction services offered by T&O.
  • Audience/Users: Homeowners and property managers evaluating services.
  • Content: Descriptions of roofing, siding, exterior remodeling, interior remodeling, general home repairs, and other projects.
  • Data Entry: No data entry requested on this page.
  • Validations: Not applicable.
  • Buttons/Hyperlinks/Dropdowns: Links to contact page and to specific project examples.
  • Actions: jQuery UI tabs or an accordion are used to switch between service categories (Roofing, Siding, Exterior Remodeling, Interior Remodeling, Other). The content for each category appears without reloading the page.
  • Special Notes: This page demonstrates the required jQuery UI widget and helps users quickly scan the services that apply to them.

Projects Page (projects.html)

  • Purpose: Showcase completed roofing, siding, exterior remodeling, and interior remodeling projects.
  • Audience/Users: Prospective clients comparing examples of T&O’s work.
  • Content: Photo gallery of completed projects categorized by service type, with short descriptions and locations.
  • Data Entry: No user-entered data on this page.
  • Validations: Not applicable.
  • Buttons/Hyperlinks/Dropdowns: Dropdown filter to select project type (Roofing, Siding, Exterior Remodeling, Interior Remodeling); clickable thumbnails that open larger images.
  • Actions: JavaScript filters the displayed projects by service type; clicking a thumbnail opens a larger view in a lightbox or modal window. AJAX is used to load project details from a static JSON file and dynamically generate the gallery.
  • Special Notes: This page is the primary location for interactive images and AJAX functionality.

About Page (about.html)

  • Purpose: Introduce T&O, its background, and its owner, Rutilo De La Luz.
  • Audience/Users: Visitors who want to understand the experience and values behind the business.
  • Content: Company story, mission statement, background on Rutilo, and a brief overview of expertise in roofing, siding, exterior remodeling, interior remodeling, and general construction.
  • Data Entry: No data entry on this page.
  • Validations: Not applicable.
  • Buttons/Hyperlinks/Dropdowns: Links back to Services, Projects, and Contact pages.
  • Actions: An optional accordion separates content into sections such as “Our Story,” “Our Work Philosophy,” and “Why Choose T&O.” Users can expand and collapse sections for easier reading.
  • Special Notes: This page focuses on building trust and credibility with potential customers.

Contact & Request Quote Page (contact.html)

  • Purpose: Allow visitors to request a quote for roofing, siding, exterior remodeling, interior remodeling, or general construction projects and to contact T&O.
  • Audience/Users: Homeowners, landlords, and other prospective clients ready to inquire about services.
  • Content: Contact information and a quote request form with fields for name, email, phone, service type, project location, budget range, and project description.
  • Data Entry: Yes — users will complete the quote request form.
  • Validations: Required fields (name, email, service type, and project description); email format validation and a minimum character count for the project description.
  • Buttons/Hyperlinks/Dropdowns: Submit and reset buttons; dropdown lists for service type and budget range; navigation links to the rest of the site.
  • Actions: When the user clicks Submit, JavaScript validates the form. If any required fields are missing or invalid, inline error messages are displayed. If all fields are valid, a confirmation message and a summary of the entered information appear on the page. No data is stored in a database; all behavior is client-side only.
  • Special Notes: This page demonstrates form handling and validation without any server-side CRUD operations.

Dynamic Functionality

The T&O website will include several pieces of dynamic functionality implemented with JavaScript, jQuery, and jQuery UI. These features are designed to improve usability and demonstrate interactive web design techniques that do not require server-side data storage.

  • Interactive Project Gallery (projects.html): A grid of project thumbnails can be filtered by service type, and each thumbnail opens a larger image and more details in a lightbox or modal overlay. Project data (title, service type, description, image path) will be stored in a static JSON file and loaded using AJAX.
  • Service Tabs or Accordion (services.html): jQuery UI tabs or an accordion will organize roofing, siding, exterior remodeling, and interior remodeling content. This allows visitors to switch between service categories without reloading the page.
  • About Page Accordion (about.html): An optional accordion will present “Our Story,” “Our Work Philosophy,” and “Why Choose T&O” as expandable sections, making it easier to scan and read.
  • Contact Form Validation (contact.html): JavaScript will validate required fields and email format before allowing the quote request form to be submitted. If the form is valid, a confirmation message and a summary of the user’s input will be displayed on the page.
